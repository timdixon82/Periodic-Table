# Project Accessibility: Periodic-Table

This project meets WCAG 2.2 at AAA, interpreted in the global wiki's `accessibility.md`.

This page records only what is specific to Periodic-Table: its accessibility notes, and a pointer to its exceptions.

## Project-specific notes

A periodic table is, by tradition, a visual grid. Presenting it to a screen-reader user at AAA needs deliberate care: a meaningful reading order, a clear table or grid structure with row and column headers, accessible names for every element cell, and a non-visual way to convey the colour-coded element categories. Carol's baseline audit and the project's decision records record how this is handled.
